// 关键：把 baseURL 改成千聚提供的国内直连地址

    try {
        console.log("正在向GPT-5发送请求...");
        
        // 调用千聚API，请求GPT-5模型
        // 注意：这里的 model 字段，你可以直接用 'gpt-4o' 或者 'gpt-4-turbo' 等

        const completion = await openai.chat.completions.create({
            model: 'gpt-4o', // 你可以换成任何支持的最强模型
            messages: [
                {
                    role: 'system',
                    content: '你是一个有用的AI助手。'
                },
                {
                    role: 'user',
                    content: '请用中文告诉我，千聚这个平台怎么让国内开发者安全地调用GPT-5？'
                }
            ],
            max_tokens: 1000,
            temperature: 0.7,
        });

        // 输出返回的结果
        console.log('GPT-5回复成功！');
        console.log('回复内容：', completion.choices[0].message.content);
        console.log('Token消耗：', completion.usage.total_tokens);

    } catch (error) {
        console.error('调用API时出错了：', error.message);
        // 如果出错，很可能是网络问题或API Key不对
        // 检查你的网络是否能正常访问 www.qianjuai.com
    }
}

## 国内服务器部署全流程：小白也能轻松搞定

代码在你自己电脑上跑通了，现在要把它部署到国内服务器上，比如阿里云、腾讯云、华为云。这套流程保证你安全不封号。

### 第一步：准备一台国内服务器

买一台最低配的云服务器就行，比如2核2G，安装Ubuntu 20.04或CentOS 7以上的系统。国内服务器直接就能用，不需要挂任何代理。

### 第二步：远程连接服务器

用SSH连接你的服务器，推荐用Xshell或FinalShell，或者直接在命令行里：

bash
ssh root@你的服务器IP


### 第三步：安装Node.js

在服务器上，执行以下命令安装最新版Node.js：

bash
curl -sL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs   # 如果是Ubuntu
# 或者如果是CentOS，用 yum install nodejs -y


验证安装：

bash
node -v   # 应该显示 v20.x.x
npm -v


### 第四步：把代码传到服务器

你可以用Git拉取，或者用scp命令直接把项目文件夹传到服务器。

假设你的项目文件夹叫 `my-gpt5-app`，在服务器上:

bash
mkdir /var/www/my-gpt5-app


然后在本机上传：

bash
scp -r my-gpt5-app/* root@你的服务器IP:/var/www/my-gpt5-app/


### 第五步：在服务器上安装依赖并运行

进入服务器上的项目目录：

bash
cd /var/www/my-gpt5-app
npm install


然后运行：

bash
node gpt5-test.js


看到GPT-5的回复了吗？一切都在国内网络上完成，没有出海的痕迹，你的服务器和账号100%安全。

### 第六步：让它一直运行（推荐PM2）

测试没问题后，用PM2让程序后台运行：

bash
npm install -g pm2   # 安装PM2
pm2 start gpt5-test.js --name gpt5-app   # 启动并命名为gpt5-app
pm2 save   # 保存进程列表
pm2 startup   # 设置开机自启


现在你的GPT-5应用就24小时跑在你的国内服务器上了。
