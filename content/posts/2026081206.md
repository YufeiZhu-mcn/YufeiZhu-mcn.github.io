## 方式一：官方直连（最折腾、最贵、最不推荐）

先说说大家第一反应想到的：直接去xAI官方网站搞一个API Key，然后用Python的`openai`库去调。

**优点**：数据路径最短，最“正统”。

**缺点**：麻烦到爆炸。你需要海外信用卡（很多国内双币卡还不行），VPN随时掉线，API Key动不动就卷款跑路。更别说xAI定价本身就高，对国内开发者来说既不友好也不划算。

**代码示例**：


import openai

# 官方直连配置
openai.api_key = "你的xAI_API_KEY"
openai.api_base = "https://api.x.ai/v1"

response = openai.ChatCompletion.create(
    model="grok-4",
    messages=[
        {"role": "user", "content": "用Python写一个快速排序"}
    ]
)

print(response.choices[0].message.content)


**结论：** 除非你钱多不怕折腾，否则直接跳过。

import openai

API_KEY = "YOUR_QIANGU_API_KEY"

def call_grok4_safely(user_input: str, max_tokens: int = 1024, retries: int = 3) -> str:
    """
    最省钱的Grok4调用方式。
    1. 超时自动重试
    2. 限制Token用量防浪费
    3. 兼容OpenAI SDK
    """
    openai.api_base = API_BASE
    openai.api_key = API_KEY

    for attempt in range(retries):
        try:
            response = openai.ChatCompletion.create(
                model="grok-4",
                messages=[
                    {"role": "user", "content": user_input}
                ],
                max_tokens=max_tokens,  # 控制输出Token长度，变相省钱
                temperature=0.7,
                timeout=30  # 超时保护
            )
            return response.choices[0].message.content
        except Exception as e:
            if attempt < retries - 1:
                wait_time = 2 ** attempt
                print(f"第{attempt+1}次调用失败，{wait_time}秒后重试...错误：{e}")
                time.sleep(wait_time)
            else:
                raise Exception(f"调用Grok4失败，已重试{retries}次：{e}")

# 使用
result = call_grok4_safely("写一个Python装饰器，计算函数执行时间")
