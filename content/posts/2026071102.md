### Step 4：流式调用的避坑点

如果接的是对话流式UI，代码改成：

java
// 开启stream
ChatCompletionRequest request = ChatCompletionRequest.builder()
        .model("gpt-5.2-codex")
        .messages(messages)
        .stream(true) // 关键：开启流式
        .maxTokens(4096)
        .build();

// 使用流式监听
service.streamChatCompletion(request)
    .blockingForEach(chunk -> {
        if (chunk.getChoices() != null && !chunk.getChoices().isEmpty()) {
            String delta = chunk.getChoices().get(0).getMessage().getContent();
            if (delta != null) {
                System.out.print(delta); // 逐Token输出
            }
        }
    });
