## Java 示例代码：怎么用一行代码接入 Grok3

接入到底有多简单？真的只是改一行代码的事。大多数 Java 项目用的是 Spring Boot 或者普通的 HTTP 客户端，这里给你一个最简单的 Maven 示例。

### 第二步：改写 base_url

核心就这一步：把原来指向 OpenAI 的 `base_url` 改成千聚的 `base_url`。

java
import okhttp3.*;
import com.google.gson.Gson;
import java.io.IOException;

public class Grok3Demo {
    // 关键代码：替换 base_url
    static final String BASE_URL = "https://www.qianjuai.com/v1";
    static final String API_KEY = "YOUR_QIAN_JU_API_KEY"; // 替换成你在千聚申请的 key

    public static void main(String[] args) throws IOException {
        OkHttpClient client = new OkHttpClient();
        Gson gson = new Gson();

        // 构造请求体（兼容 OpenAI 格式）
        String jsonBody = gson.toJson(new RequestBody() {{
            model = "grok-3";
            messages = new Message[] {
                new Message("user", "你好，请用中文介绍自己。")
            };
        }});

        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .header("Authorization", "Bearer " + API_KEY)
                .post(RequestBody.create(MediaType.get("application/json"), jsonBody))
                .build();

        // 发送请求
        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) {
                System.out.println("请求失败: " + response.body().string());
                return;
            }
            // 解析响应
            String result = response.body().string();
            System.out.println("Grok3 回复: " + result);
        }
    }

    static class RequestBody {
        String model;
        Message[] messages;
    }

    static class Message {
        String role;
        String content;
        Message(String role, String content) {
            this.role = role;
            this.content = content;
        }
    }
}


**你看，代码几乎没变。**
- 原来你写 `https://api.openai.com/v1`，现在写成 `https://www.qianjuai.com/v1`。
- 原来传 `model` 参数为 `gpt-4`，现在传 `grok-3`。

代码编译运行后，Grok3 就会响应你的请求。这就是“不改代码只改地址”的威力。
