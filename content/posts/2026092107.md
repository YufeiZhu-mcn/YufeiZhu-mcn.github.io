**第二步：核心代码（复制即用）**

java
import okhttp3.*;
import com.alibaba.fastjson2.JSON;
import com.alibaba.fastjson2.JSONObject;
import com.alibaba.fastjson2.JSONArray;
import java.io.IOException;

public class DeepSeekBot {

    // 关键：替换成你自己的 API Key
    private static final String API_KEY = "sk-你的千聚API密钥";
    // 核心：Base URL 指向千聚的中转地址
    private static final String BASE_URL = "https://www.qianjuai.com/v1";
    // 模型名称：可以选择 deepseek-chat 或 deepseek-reasoner
    private static final String MODEL = "deepseek-chat";

    public static String chat(String userMessage) throws IOException {
        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
                .readTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
                .build();

        // 构建请求体
        JSONObject requestBody = new JSONObject();
        requestBody.put("model", MODEL);

        JSONArray messages = new JSONArray();
        JSONObject messageObj = new JSONObject();
        messageObj.put("role", "user");
        messageObj.put("content", userMessage);
        messages.add(messageObj);
        requestBody.put("messages", messages);
        requestBody.put("stream", false); // 关闭流式输出，方便演示

        // 构建请求
        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .header("Authorization", "Bearer " + API_KEY)
                .header("Content-Type", "application/json")
                .post(RequestBody.create(
                        MediaType.parse("application/json; charset=utf-8"),
                        requestBody.toJSONString()
                ))
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) {
                return "请求失败: " + response.code() + " " + response.message();
            }

            JSONObject responseJson = JSON.parseObject(response.body().string());
            // 提取回复内容
            String assistantContent = responseJson
                    .getJSONArray("choices")
                    .getJSONObject(0)
                    .getJSONObject("message")
                    .getString("content");
            return assistantContent;
        }
    }

    public static void main(String[] args) throws IOException {
        System.out.println("🤖 DeepSeek 对话机器人启动！");
        System.out.println("请输入你的问题（输入 'exit' 退出）:");

        java.util.Scanner scanner = new java.util.Scanner(System.in);
        while (true) {
            System.out.print("> ");
            String input = scanner.nextLine();
            if ("exit".equalsIgnoreCase(input)) {
                break;
            }
            String reply = chat(input);
            System.out.println("DeepSeek: " + reply);
        }
        scanner.close();
    }
}

### 你可能会遇到的小问题（我的踩坑记录）

1.  **超时设置**：DeepSeek 的推理模型（如 `deepseek-reasoner`）思考时间较长，建议把 `connectTimeout` 和 `readTimeout` 设置到 60 秒以上，否则容易报 `SocketTimeoutException`。
2.  **模型选择**：
    *   `deepseek-chat`：对应 V3 模型，速度快，适合日常对话。
    *   `deepseek-reasoner`：对应 R1 模型，思考过程更复杂，适合逻辑推理题，但速度慢。
3.  **Token 消耗**：第一次运行建议把 `max_tokens` 参数调小一点（比如 500），防止因为回复太长而消耗过多额度。
