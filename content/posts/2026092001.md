## 怎么用 Java 接入 Kimi？不用再搜教程了

你搜“Kimi统一接入Java示例”，90%的教程会教你做以下几步：

### 第一步：改 base_url

java
String baseUrl = "https://www.qianjuai.com/v1"; // 替换为你的API网关
String apiKey = "sk-你的key";


### 第二步：写一个简易的 HTTP 客户端

java
import java.net.http.*;
import java.net.URI;
import java.net.http.HttpRequest.BodyPublishers;

HttpClient client = HttpClient.newHttpClient();

// 构建请求体（兼容 OpenAI 格式）
String requestBody = """
{
    "model": "kimi", 
    "messages": [
        {"role": "user", "content": "用Kimi API写一个Java示例"}
    ]
}
""";

HttpRequest request = HttpRequest.newBuilder()
    .uri(URI.create(baseUrl + "/chat/completions"))
    .header("Content-Type", "application/json")
    .header("Authorization", "Bearer " + apiKey)
    .POST(BodyPublishers.ofString(requestBody))
    .build();

client.sendAsync(request, HttpResponse.BodyHandlers.ofString())
    .thenApply(HttpResponse::body)
    .thenAccept(System.out::println)
    .join();


**就这几行代码**，这就是所谓的“Kimi统一接入”。那些卖大几千块的教程核心内容不超过这 10 行。

**关键点：** 不需要引进 Kimi 特有的 SDK，不需要改参数结构，直接用 OpenAI 标准的 JSON 格式。把 `model` 换成 `"kimi"` 或者其他模型名（`gpt-4`、`claude-3.5` 都行）就能切换不同的模型。
