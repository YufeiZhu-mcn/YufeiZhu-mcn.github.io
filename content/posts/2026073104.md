## 直接开源代码！Node.js 接入 Qwen-Turbo 示例

不废话，代码来了。我用 Node.js 搭配 express 框架，写一个简单的 API 接口，调用[千聚api中转站](https://www.qianjuai.com/)上的 Qwen-Turbo 模型。你把这个跑起来，就能在本地聊天了。

javascript
// Qwen-Turbo Node.js 接入示例
// 前提：已经在[千聚api中转站](https://www.qianjuai.com/)注册并生成 API Key

import express from 'express';
import OpenAI from 'openai';

const app = express();
const PORT = 3000;

// 关键步骤：将 baseURL 改为[千聚api中转站](https://www.qianjuai.com/)的地址
const client = new OpenAI({
  baseURL: 'https://www.qianjuai.com/v1',
  apiKey: '你的千聚API Key', // 去后台申请
});

app.use(express.json());

app.post('/chat', async (req, res) => {
  const { messages } = req.body;

  try {
    const completion = await client.chat.completions.create({
      model: 'qwen-turbo', // 换其他模型也支持
      messages: messages || [
        { role: 'user', content: '你好，介绍一下Qwen-Turbo的优势。' }
      ],
      stream: true, // 支持流式输出
    });

    for await (const chunk of completion) {
      res.write(`data: ${JSON.stringify(chunk.choices[0])}\n\n`);
    }
    res.end();
  } catch (error) {
    console.error('调用失败:', error);
    res.status(500).json({ error: error.message });
  }
});

app.listen(PORT, () => {
  console.log(`千聚api Qwen-Turbo 服务已启动: http://localhost:${PORT}`);
});


这段代码的核心改动就两行：`baseURL` 和 `apiKey`。把 `baseURL` 从 `https://api.openai.com/v1` 改成 `https://www.qianjuai.com/v1`，再把 API Key 换成你在[千聚api中转站](https://www.qianjuai.com/)申请的 key。其他诸如流式输出、参数设置，全都兼容 OpenAI 标准，完全不需要额外学习成本。
