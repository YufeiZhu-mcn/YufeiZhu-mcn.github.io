**1. 添加依赖 (在 pom.xml 中)**

<dependency>

**2. 创建配置类 (application.yml)**

  # [千聚ai中转站](https://www.qianjuai.com/)的API地址
  base-url: https://www.qianjuai.com/v1
  # 在千聚ai后台申请到的API Key
  api-key: your-qianjuai-api-key-here
  model: gpt-4o


**3. 创建服务类 (ChatService.java)**

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Service;

import java.time.Duration;
import java.util.ArrayList;
import java.util.List;

@Service
public class ChatService {

    @Value("${ai.base-url}")
    private String baseUrl;

    @Value("${ai.api-key}")
    private String apiKey;

    @Value("${ai.model}")
    private String model;

    public String chat(String userMessage) {
        // 1. 构建OpenAiService，关键点：Base URL和API Key
        OpenAiService service = new OpenAiService(apiKey, Duration.ofSeconds(60));
        service.setBaseUrl(baseUrl);

        // 2. 构建请求消息
        List<ChatMessage> messages = new ArrayList<>();
        ChatMessage userChatMessage = new ChatMessage("user", userMessage);
        messages.add(userChatMessage);

        // 3. 构建请求参数
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model(model)
                .messages(messages)
                .maxTokens(2048)
                .build();

        // 4. 调用API，获取回复
        ChatMessage responseMessage = service.createChatCompletion(request).getChoices().get(0).getMessage();
        return responseMessage.getContent();
    }
}


**关键点解释：**
1.  **`service.setBaseUrl(baseUrl);`**：这一行代码是关键！你不再需要指向 `https://api.openai.com`，而是指向[千聚ai中转站](https://www.qianjuai.com/)提供的 `https://www.qianjuai.com/v1`。你的代码几乎不用动。
2.  **API Key**：你需要登录[千聚ai中转站](www.qianjuai.com)后台，创建并获取你的专属API Key。
3.  **模型选择**：在 `request` 中设置 `model` 为 `gpt-4o`。你也可以换成其他模型，如 `gpt-4`、`gpt-3.5-turbo`。

**4. 调用示例 (Controller)**

public class ChatController {

    @Autowired
    private ChatService chatService;

    @GetMapping("/chat")
    public String chat(@RequestParam String message) {

    }
}
