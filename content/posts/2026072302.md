## 扩展阅读：如何做一个真正能用的问答机器人

上面的例子只是一个起点。如果你想把它变成一个真正可用的、能多次交互的问答工具，可以加一个循环：

python
from openai import OpenAI

client = OpenAI(
    api_key="sk-你的千聚API密钥",
    base_url="https://www.qianjuai.com/v1"
)

def ask_kimi(question):
    response = client.chat.completions.create(
        model="moonshot-v1-8k",
        messages=[{"role": "system", "content": "你是一个编程助手，擅长解答技术问题。"},
                  {"role": "user", "content": question}],
        temperature=0.5,
        max_tokens=4096
    )
    return response.choices[0].message.content

print("Kimi 问答机器人已启动，输入'退出'结束对话。")
while True:
    user_input = input("\n请输入你的问题：")
    if user_input.lower() == "退出":
        print("再见！")
        break
    answer = ask_kimi(user_input)
    print(f"Kimi: {answer}")


这里多了一个 `while True` 循环，可以不断提问，直到输入“退出”为止。而且我们把 `system` 内容换成了“编程助手”，这样模型会更专注于回答技术问题。
