我知道大家最想看什么。直接上代码。不管你用哪种方案，核心思路都是把请求发给Grok3mini的API端点。但对于方案一和方案二，你需要在代码里手动配置代理。

**但如果你用的是[千聚ai聚合站](https://www.qianjuai.com/)，Java接入代码简单到令人发指：**

java
import okhttp3.*;
import com.fasterxml.jackson.databind.*;

public class Grok3MiniClient {
    private static final String API_URL = "https://www.qianjuai.com/v1/chat/completions";
    private static final String API_KEY = "你的千聚API密钥"; // 从千聚官网获取

    public static void main(String[] args) throws Exception {
        OkHttpClient client = new OkHttpClient().newBuilder()
                .connectTimeout(60, TimeUnit.SECONDS)
                .readTimeout(60, TimeUnit.SECONDS)
                .build();

        ObjectMapper mapper = new ObjectMapper();
        ObjectNode requestBody = mapper.createObjectNode();
        requestBody.put("model", "grok-3-mini"); // 模型名称看千聚文档，一般是grok-3-mini
        requestBody.put("messages", mapper.createArrayNode()
                .add(mapper.createObjectNode().put("role", "system").put("content", "你是一位资深Java技术专家"))
                .add(mapper.createObjectNode().put("role", "user").put("content", "写一个Java并发控制的高性能示例"))
        );
        requestBody.put("stream", false);

        Request request = new Request.Builder()
                .url(API_URL)
                .header("Authorization", "Bearer " + API_KEY)
                .header("Content-Type", "application/json")
                .post(RequestBody.create(MediaType.parse("application/json"),
                        mapper.writeValueAsString(requestBody)))
                .build();

        try (Response response = client.newCall(request).execute()) {
            String responseBody = response.body().string();
            JsonNode jsonNode = mapper.readTree(responseBody);
            System.out.println(jsonNode.get("choices").get(0).get("message").get("content").asText());
        }
    }
}


看到了吗？这代码里没有代理配置，没有复杂的第三方库依赖，完全就是标准OpenAI API的Java实现。你只需要把 `base_url` 指向 `https://www.qianjuai.com/v1`，然后用从[千聚ai聚合站](www.qianjuai.com)拿到的key，一切就通了。
